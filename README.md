# endless-progress-quest
纯文字放置挂机游戏｜暗黑风
无尽的进度条 —— 战斗版 v1.0.0
无广告｜无内购｜完全免费｜单机本地运行

一、玩法介绍
全程自动挂机战斗，全程无需任何操作
开局随机六大属性，一旦确定终身无法更改
后期战力不足、打不动怪属于正常设定
号弱直接重开，刷出天胡开局即可一路平推

二、属性说明
力量：作为主属性时大幅提升普攻伤害，同时增加背包负重
体质：提升血量上限与生命恢复速度
敏捷：作为主属性时提升普攻伤害，同时增加角色防御
智力：作为主属性时提升普攻伤害，强化法术伤害，并提高蓝量上限
悟性：提升高阶技能领悟概率，加快魔法值恢复
魅力：商店买药享受折扣，同时提高装备掉落爆率

三、怪物等级机制
超越自身 3 级 = 普通喽啰怪
超越自身 6 级 = 高阶精英怪
超越自身 9 级 = 终极暗金 BOSS
越级挑战奖励更丰厚，但风险更高，角色死亡将自动回城复活

四、重要提示
1. 本程序基于开源游戏《无尽的进度条》二次开发，无微软数字签名，部分杀毒软件可能误报病毒，属正常现象，可放心使用。
2. 游戏为纯单机本地运行，不联网，不收集任何个人信息。
3. 存档文件存于程序同级目录，请勿随意删除，否则存档将永久丢失。
4. 支持窗口最小化后台挂机，不影响电脑正常使用。
